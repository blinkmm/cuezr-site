First-launch notes

Mac: The app is signed with a Developer ID but not yet notarized. On first launch, right-click the app and choose Open to bypass the Gatekeeper warning. After that it opens normally.

Windows: Windows SmartScreen may show a warning on first launch. Click More info → Run anyway.

Both: Click Allow when prompted for network access — CUEZR needs UDP ports for OSC data from OSCPoint.
